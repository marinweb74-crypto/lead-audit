# Agent Runtime

Рабочая директория для обмена данными между агентами.

## Структура

- `shared/` — рабочие данные (leads.json, audits.json, outreach.json, логи)
- `state/` — план и статус pipeline (plan.md, status.md)
- `messages/` — handoff-сообщения между агентами
- `outputs/` — финальные результаты (report.md, review-report.md, PDF-файлы)
